# 표류 드리프터 선행사례 연구 — 기구제작·운영실험·데이터기록 3축 분석

> **한 줄 요약:** NOAA GDP(위성·수중 15m 드로그·40년 표준)부터 OpenMetBuoy(오픈하드웨어 위성 부이), Duncan·Merlino·Kang·한강하구(PET·셀룰러 하천 드리프터), PlasticPirates·Precious Plastic(시민과학 거버넌스)·홍콩 Marine Litter Detective(학생 110대·11학교)까지 8개 선행사례를 **기구체 제작 / 운영 실험 / 데이터 기록** 세 축으로 해부해, Friendly Floaty(셀룰러 LTE·PET병·미회수 손실 감수·OpenDrift 보정)의 설계·운영·FF-ID 스키마에 그대로 이식할 교훈만 추린다.

## 0. 방법과 정직 고지

- 본 문서의 모든 수치에는 신뢰도 태그를 붙인다.
  - **[출처확인]** — canonical 문헌에서 직접 확인, 그대로 인용 가능.
  - **[단일출처·주의]** — 인용 가능하나 단일 문헌 의존, "○○ 논문 기준·교차검증 없음" 병기.
  - **[출처불명·표시필수]** — 뒷받침 공개 URL 없음. 단정 인용 금지, "출처 미확인" 딱지 필수.
- **미확인 항목(원가·통신방식·기구체 재료 등)은 임의 수치로 메우지 않고 "자료 없음"으로 둔다.**
- **배터리 수명의 "연 단위" 값(예: Duncan 6xAA 약 9년, OMB 약 4.5개월)은 대기전류/샘플링 기준 이론적 외삽 추정치이며 실측 운용수명이 아니다.**
- **폐기 처리한 수치:** Merlino(Case 4) 초록의 "2,845 km / 94일"은 (a)같은 논문 본문 실측 최장(852.59 km)과 3.3배 불일치, (b)Duncan(Case 3) B2 해상 궤적 수치와 완전히 동일 → 단일 드리프터 최대 성능으로 **절대 인용 금지·폐기**. 이 우연한 동일값으로 두 사례를 교차인용하지 않는다.
- **OpenDrift 귀속 정정:** Kang 2025(Case 5)는 OpenDrift를 사용하지 않았다(자체 VSC + windage 계수 분석).

## 1. 사례별 카드

### Case 1 — NOAA Global Drifter Program (SVP 드리프터)
**한 줄:** 40년 글로벌 표준. 구형 부표를 수심 15m holey-sock 드로그에 매달아 항력비 약 40으로 "물을 따라가게" 설계, 수천 기 방류·엄격 QC·안정적 메타데이터 디렉토리를 갖춘 데이터 스키마의 참조 원형 — 단, 목적(수중 해류 측정)이 FF(수면 쓰레기 추적)와 정반대다.

- **제작:** 민간사(Clearwater·Pacific Gyre·Technocean·Metocean·Marlin-Yug)가 DBCP 규격에 맞춰 제작. 부표 지름 30.5~40cm 구형, 초기 유리섬유 0.3~0.4cm 두께→현재 대부분 ABS. 드로그 중심 15m, 7섹션×92cm. 하부 습식면 산화구리 방오도료, 자석 풀 활성화. [출처확인]
- **운영:** 선미에서 투하, 항해 중 가능(최대 25노트 화물선 방류 실적). 종이테이프·수용성 종이관이 녹아 드로그 침강까지 약 1시간. **회수하지 않음** — 사멸·좌초·시민 습득·송신중단까지 자유표류, 유닛별 death code 기록. [출처확인]
- **데이터:** 고전형 Argos(401.650MHz) 도플러 측위, 정확도 150~1000m. 센서 90초 간격 7~10샘플 평균. QC = Hansen&Poulain 2단계(전진·후진 유한차분 양쪽 모두 4σ 초과 시 bad). Argos를 6시간 격자 크리깅, 신형 시간별 GPS(약 10m)/Iridium NetCDF. 메타데이터 디렉토리 = 방류/종료 시각·위경도·드로그상실일·death type·부이형·제조사·WMO/PTT ID, 수정 율리우스일(1979-01-01=1일). CF 규약 contiguous ragged array NetCDF·ERDDAP·NCEI·AWS 배포. [출처확인]
- **핵심수치:**
  - 항력비 설계목표 약 40(측정 37.5~45.9), 다운윈드 슬립 0.7 → 약 8.6 cm/s(드로그 유무, 풍속 10m/s당) [출처확인]
  - Argos 실측 RMS 630m(동서)/270m(남북), 측위 6~9회/일→16~20회/일(다위성) [출처확인]
  - **방류 규모 2003–04: 658기/년(R/V 440 + 화물선 201 + 항공 17)** — Lumpkin&Pazos 단일 챕터 기준 [단일출처·주의]
  - **유닛 원가 5,475 USD(1993, 2003$)→1,750 USD(2004)** — 동일 단일 챕터 [단일출처·주의]
  - 수명 약 400일(제조사 추정), 송신기 반감기 291일(1998)→485일(2001) [단일출처·주의]
- **출처:** [Lumpkin & Pazos 기술챕터](https://www.aoml.noaa.gov/phod/docs/LumpkinPazos.pdf) · [GDP Data FAQ](https://www.aoml.noaa.gov/phod/gdp/data_faq.php) · [Hourly Drifter Data](https://www.aoml.noaa.gov/phod/gdp/hourly_data.php) · [Elipot et al. 2016](https://agupubs.onlinelibrary.wiley.com/doi/full/10.1002/2016JC011716)
- **자료 없음:** 밸러스트 질량, 미니형 하우징 두께, 배터리 용량, 신형 GPS/Iridium 샘플링 주기, 드로그상실 판정 수치 임계값.

### Case 2 — OpenMetBuoy-v2021 (OMB, Rabault et al. 2022)
**한 줄:** 위성(Iridium) 오픈소스 표류·파랑 부이. MIT 라이선스로 설계·펌웨어·후처리 코드 전면 공개, 통합 캐리어 보드 전략으로 조립 0.5시간 — FF의 벤치마크 원형이나 통신이 위성이라 전력·통신 설계가 FF의 셀룰러와 갈린다.

- **제작:** SparkFun Artemis Global Tracker(MCU+GNSS+Iridium 통합) 1보드에 납땜 2곳·Qwiic 플러그로 조립. 9dof = ST ISM330DHCX + LIS3MDL. 전원 SAFT LSH20 리튬 D셀 2개 병렬 + Pololu 3.3V 벅. 부력=스티로폼(덕트테이프 감아 마모방지), 바닥 킬=볼트 스크랩, 에폭시 밀봉. IP68 12×12×9cm. [출처확인]
- **운영:** 검증 배치 4건(스발바르 해빙 11개월+·도쿄대 수조·NABOS 중앙북극에서 상용 SOFAR Spotter 동시방류·카리브해) → 2022 스발바르 주변 **70개 궤적+** 다기관. 표류형 미회수. [출처확인]
- **데이터:** GNSS 30분·파랑 스펙트럼 2시간. 파랑 = 원시 약 800Hz→3σ 100Hz 평균→Kalman(NED)→10Hz 수직가속도→20분 Welch(21세그·75%중첩), 0.05~0.5Hz. Iridium 9603 양방향(원격 샘플링율 변경), 바이너리 버퍼·무손실 재전송, Python 디코더 동봉. 3σ QC. SOFAR Spotter와 매우 양호 일치. [출처확인]
- **핵심수치:**
  - **BOM 부품비 562 USD(2021-10)** [단일출처·주의]
  - **6개월 총소유비용 약 1,200 USD(약 53%가 Iridium 통신료)** [단일출처·주의]
  - MCU Ambiq Apollo3 Blue, 보드 약 $375 [단일출처·주의]
  - **평균 18.7 mW, D셀 2개로 기본 샘플링 시 약 4.5개월** — 대기·샘플링 기준 추정(실측 아님) [단일출처·주의]
  - 실내 노이즈 하한 약 0.1cm, 조립 0.5시간/대, 무게 0.3~0.5kg [출처확인]
  - ※ "노르웨이 대량주문 약 650 USD/대가 BOM 562 USD보다 비쌈"은 반직관 → **인용 자제**.
- **출처:** [Geosciences 12(3):110](https://www.mdpi.com/2076-3263/12/3/110) · [arXiv:2201.08384](https://arxiv.org/abs/2201.08384) · [GitHub OpenMetBuoy-v2021a](https://github.com/jerabaul29/OpenMetBuoy-v2021a)
- **자료 없음:** GNSS 정확 모델명, 국제 메타데이터 표준 준수 여부, GNSS 수평 정확도(m), 2022 방류 정확 대수.

### Case 3 — Duncan et al. 2020 "Message in a bottle" (PLOS ONE)
**한 줄:** PET병에 담은 오픈소스 GPS 트래커 25기(GSM 10 + Argos 위성 15)를 갠지스강·벵골만에 방류 — 강에서는 약 40%가 수십 km 내 좌초, FF의 직계 조상 격.

- **제작:** 500mL PET병 개조 + CNC ABS 인클로저. 방수 = 2.5mm O링(가공 홈) + 안테나 개구부 Araldite 에폭시. Arribada Horizon GPS PCB(+위성형 ARGOS ARTIC R2), Icoteq 외주. 펌웨어 GPLv3·하드웨어 CERN OHL 1.2. [출처확인]
- **운영:** Phase A GSM 10기 인도(2019-05~07), Phase B 위성 15기 방글라데시+해상 3기(2019~2020-10~12). **체계적 회수 없음**, 시민 SIM 재사용 사례. 사이클론·어업 손실. [출처확인]
- **데이터:** Phase A GPS 3시간(30초 취득)·GSM→AWS(외곽 수신 불량). Phase B 4시간+위성통과(고도≥40°)·Argos. GPS마다 시계재동기. QC = Excel→ArcMap 10.7(WGS84)→R 3.6.2 sf. [출처확인]
- **핵심수치:**
  - 총 25기, 무게 약 130g(안테나 수면 위 유지) [출처확인]
  - **좌초율 약 40%(평균 72.2km 이동 후)** [출처확인]
  - 하천 최장 609.9km/51일, 해상 평균 41.6일/1,211.7±846.2km [출처확인]
  - **최장 단일 궤적 2,845km/94일(B2, 해상)** — 논문 본문 확인값. Merlino 초록 동일값과 우연 일치 → 교차인용 금지 [출처확인·주의]
  - idle 38 µA → **6xAA 약 9년+ 대기** = 대기전류 기준 외삽(실운용 수명 아님) [단일출처·주의]
  - **유닛 원가: 논문 미공개** [자료 없음]
- **출처:** [PLOS ONE](https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0242459) · [PMC7710111](https://pmc.ncbi.nlm.nih.gov/articles/PMC7710111/)
- **자료 없음:** 유닛 원가·전체 BOM, Horizon 리비전, 제작 대수, 조립 주체, AA 외 밸러스트.

### Case 4 — Merlino et al. 2023 "Marine Litter Tracking System" (Arno강, 고교생 제작, Sensors 23(2):935)
**한 줄:** GSM/GPS 저가 드리프터를 고교생이 제작(목판형·절단 페트병형), Arno 하구에서 14기·5회 방류, 시간별 GSM 측위 + SD 폴백 + QR 기반 시민 회수.

- **제작:** 목판형 약 38×38×4cm 방수도장 나무, 병형 2~5L 절단 플라스틱 수지 재밀봉, 상단 전자박스 약 10×10×7cm 실리콘 밀봉. **밸러스트로 몸체 약 50% 수면 아래**. 상용 Winnes TPK-905(5000mAh) 또는 자작 MADUINO SIM808. 태양광 약 5×5인치. 제작=Capellini-Sauro 고교생. [출처확인]
- **운영:** Arno 하구(43.68202N/10.26484E ±0.005°) 5회: 2021-04-16·08-09·08-12·09-15·12-16. 방류시각 11:00 CET. **회수 실시**(QR "발견하면 연락"), 시민·국립공원·경찰 잠수팀 회수·재사용. 여름 방류 수시간 좌초. [출처확인]
- **데이터:** GPS+GSM(해안 약 10~20km), 상실 시 SD 로깅 후 지연 전송. 샘플링 통상 60분. 필드 = 일시(UTC)·위경도·속도·유향·GSM신호·배터리·온도. Excel/CSV→GPX Studio. QC = SD 버퍼 포인트 시간순 재정렬. GPS 정확도 미기록. [출처확인]

- **출처:** [Sensors 23(2):935 (PMC9863889)](https://pmc.ncbi.nlm.nih.gov/articles/PMC9863889/) · [DOI 10.3390/s23020935](https://doi.org/10.3390/s23020935)
- **자료 없음:** 유닛 제작비, MADUINO 배터리 사양, 정식 메타데이터 표준명, 개별 GPS 정확도/HDOP.

### Case 5 — Kang et al. 2025 제주 4하구 IoT/셀룰러 드리프터 11기 (KJRS 41(5):911)
**한 줄:** 동아대팀이 오픈소스 MCU + 다중망 셀룰러(NB-IoT/LTE-M Cat-M1/2G) + GNSS 드리프터 11기를 제주 4하구에 우기 대조기 방류, 1시간 측위·클라우드 전송, VSC + windage 계수로 "바람이 아니라 해류가 지배"함을 실증. **FF 전자스택과 사실상 동일.**

- **제작:** "오픈소스 MCU + GNSS" + "다중망 셀룰러(NB-IoT/LTE-M Cat-M1/GPRS)" + 리튬이온. **보드/칩셋 모델명·하우징·밸러스트·부력·치수·무게·원가 미공개.** ("SIMCom 기반"은 추정으로 **논문 명시 아님**.) [일부 출처확인/상세 자료 없음]
- **운영:** 2회 캠페인, 대조기+강우 동기화. 7-13~14(5기)·8-10~11(6기), 총 11기. 4하구(천미천 E·금성천 W·효돈천 S·화북천 N). KHOA 고조 시각표 동기화. **회수 절차 없음.** 1-1호 약 9시간 외해 표류 후 송신중단. [출처확인]
- **데이터:** GNSS <2.5m(CEP 50%), 1시간 샘플링. 셀룰러 클라우드 업로드, 연결 상실 시 내부저장·복구 시 자동업로드, 전송 간 슬립. 시간순 정렬+선형보간. 3시간 평균 변위 vs 6시간별 NOMADS 풍·CMEMS 유속. [출처확인]
- **핵심수치:**
  - 총 11기, 4하구 [출처확인]
  - **GNSS <2.5m CEP·1시간** — KJRS 논문 기준 [단일출처·주의]
  - **windage α: 7월 0.012±0.002 / 8월 0.014±0.003(범위 0.002~0.027)** [단일출처·주의]
  - VSC(부이 vs 해류, 7월) 0.501±0.209 [단일출처·주의]
  - 1-1호 약 9시간 후 송신중단 [출처확인]

## 3. ★ Friendly Floaty가 배울 것

### 제작 교훈

## 4. 우리와의 차이·주의

### 4-0. 국내 선례 정면 비교 (제주·한강 vs 우리)

| 구분 | 제주 4하구(Kang 2025) | 한강하구(Do 외 2026) | 우리(Friendly Floaty) |
|---|---|---|---|
| 규모·성과 | 11대·2회차 · windage α 0.012~0.014 실측 | 49대·6개월 · **쓰레기 ~12%만 외해, 대부분 하구 1달+ 잔류** | 파일럿 5대(미실시) |
| 전자·통신 | 오픈 MCU + **다중망 셀룰러(NB-IoT/LTE-M/GPRS)** + 리튬 = 우리와 사실상 같은 계열 | 기기설계 **미공개(유료)** | 셀룰러 **Cat.1(A7670)** |
| 데이터 | 1시간·store-and-forward·자체 VSC+windage(OpenDrift 미사용) | GPS 라그랑주(미공개) | 1시간·store-and-forward·FF-ID·QC 오픈 |
| 회수 | ❌ 미회수(1-1호 9시간 뒤 두절) | ❌ 미회수(추정) | ✅ **회수-우선** |
| 개방 | ❌ 원가·하우징·밸러스트 미공개 | ❌ 유료 논문 | ✅ **전면 공개** |

- **배울 점**: store-and-forward·조석 동기 방류·windage 보정(제주) 채택 완료. "하구에 오래 잔류"(한강)는 우리 회수형 초점의 실증.
- **반면교사**: 둘 다 ①미회수(연구는 손실 감수, 우리가 그러면 오염) ②설계 비공개(재현 불가) → 우리 두 차별점(회수-우선·개방)이 이 공백을 메움.

### 4-1. 상세 차이

- **위성 vs 셀룰러.** GDP·OMB·Duncan(Phase B)은 **위성**(넓은 커버리지·비싼 통신료, OMB TCO 약 53% Iridium). FF는 **셀룰러 LTE Cat.1 bis**(저렴·고대역이나 하천 음영) → Duncan·Merlino·Kang이 store-and-forward로 해결. Kang·한강하구·Merlino가 FF의 셀룰러 국내 하구 실증 표준 관행을 뒷받침. PlasticPirates·Precious Plastic엔 통신·자동측위 선례 없음.
- **회수 여부.** GDP·Duncan·OMB 표류형·Kang·(한강하구 미기재)=**미회수**. Merlino·PlasticPirates=**회수·재사용**. FF는 회수형/소모형을 **명시적으로 결정**해 예산·라벨·비컨 설계.
- **측정 대상의 반전.** SVP는 수중 15m 드로그로 **해류를 따라간다**. FF는 **수면 쓰레기를 추적**한다. SVP 모범 다수가 반전 대상(드로그 최소·저항력비·수면 추종).
- **배터리 수명 표기.** Duncan "6xAA 약 9년", OMB "약 4.5개월"은 **대기전류/샘플링 기준 이론적 외삽**. FF 사양서에도 반드시 "대기전류 기준 추정"으로만 표기.
- **OpenDrift 귀속.** Kang 2025 미사용(VSC+windage). FF의 OpenDrift 보정은 Kang의 검증 물리(해류 지배·α 소값)를 캘리브레이션 근거로 삼되, OpenDrift 자체를 Kang에 귀속하지 않는다.
- **★제주 windage 값 참고의 타당성(정직 경계).** windage는 하천이 아니라 **부유체(물 위로 나온 정도·형상)의 성질**이다. 제주값(α 0.012~0.014)은 **저건현 부유 쓰레기 프록시 + 한국 하구**라는 같은 물리 레짐이라 **우리 저건현 PET 병의 합리적 출발값(prior)**이다. 단 ①그들의 기기 형상이 미공개라 우리 병과 정확히 같지 않고 ②단일 출처다 → **"차용한 시드, 우리 병 궤적으로 반드시 자체 보정해 대체"**로만 쓴다(calibration commons 원칙). 같은 물체가 하천을 옮길 때의 전이보다 신뢰도가 낮은, 어디까지나 자릿수 수준의 사전값이다.
- **수치 오염 경고.** "2,845km/94일"은 Merlino 초록·Duncan B2에 동일 등장하나 Merlino 본문 실측 최장은 852.59km. **단일 드리프터 최대 성능으로 인용하지 않으며** 두 사례를 교차인용하지 않는다.

### 5-3. 외해 생존기간·고장모드 (실측 종합) — "방수·전원 유지" 가정에서도

| 구분 | 수명 | 비고 |
|---|---|---|
| **DIY 셀룰러(우리 급)** | **12~36시간** | Drifter Challenge, 외해 커버리지 이탈이 지배 |
| CARTHE(저가 생분해) | 최대 ~3개월 | 양호 조건 |
| SVP($1,800 전문) | half-life ~450일 | 방오도료+15m 드로그 |

- **생물부착(biofouling)**: 생물막 5~10일(>15°C) → 따개비 유의 부착 **2~6주**(열대 2배·~0.1mm/일), 방오도료조차 28일까지. 부착 질량 >10kg/m²가 부력·흘수를 무너뜨려 침몰·안테나 침수. **작고 가벼운 드리프터일수록 부력 여유가 작아 치명적.** [일반 해양문헌 전형값·재질/수온/위도 편차 큼]
- **UV 열화**: 공기 노출 PE 8주 취성화(침수 시 물이 UV 차폐로 지연). PET 병벽 ~300µm는 수면 노출부가 수개월 내 약화 → HDPE·PC·UV안정재+두께 확보. [전형값]
- **기계**: 좌초·표착 파손, 파랑 충격, 침수 단락. 하천 좌초율 ~40%(Duncan Case 3).
- ★결론: **우리 PET Tier1의 정직한 외해 생존 = 주 단위.** 450일은 방오도료+드로그 달린 $1,800 플랫폼의 값이지 우리 값이 아니다.

### 5-4. 셀룰러 외해 한계 = "커버리지 봉투"로 고지 + 위성 폴백
- 선박 안테나(5m) 실용 ~8km. **드리프터 안테나는 수면(0.1~0.3m)** → 전파 지평선 훨씬 짧음, DIY 실측 상한 **~11km**. → 셀룰러 = 하천·하구·연안 도구, 외해는 끊긴다.
- **GSM/LTE/Cat-1 정리**: GSM=2세대(2G), LTE=4세대(4G), Cat-1=LTE 기기 성능등급(하위·IoT 충분). 외해 한계는 Cat-1의 약점이 아니라 **"기지국은 육지에 있다"는 물리** — 2G·Cat-1·5G가 동일하게 끊긴다. 오히려 Cat-1은 죽은 2G(홍콩·Merlino SIM808)를 대체하는 정답. 글로벌 IoT SIM(Soracom 등)은 최적 통신사로 로밍해 연안 커버리지를 넓히는 완화책이지 한계의 원인이 아니다.
- ★**Swarm 종료 정정**: 외해 위성 폴백으로 Swarm을 들었으나 **SpaceX 인수 후 2024말 서비스 종료** → 폐기. 살아있는 폴백 = **Iridium SBD(전지구)·Astrocast Astronode S($49)·Argos ARTIC R2(Arribada식)·신흥 위성 NB-IoT/NTN(Blues Starnote·Skylo)**.
- 실증: **Arribada Horizon = Nat Geo "Sea to Source: Ganges"(= Duncan Case 3)** = 몬순 전 셀룰러 단기 고빈도 + 몬순 후 Argos 위성 저빈도 외해 장거리 = "연안=셀룰러, 외해=위성" 이원화의 실증. (외해 장거리 수치는 §0 폐기값 규율 유지 — 헤드라인 숫자로 쓰지 않음.)
- Chao Phraya(우리 쌍둥이) 구체화: The Ocean Cleanup+Chula GPS 드리프터 **n=63, 5분·~4m**.

### 5-5. 보드 선택 검증 — A7670 Cat-1 vs SIM7000/7080, 바꿔야 하나? → **현행 유지**
- **A7670(Cat-1)**: 넓은 LTE-FDD 커버리지(특수 프로비저닝 불필요)·처리량 여유. 단 송신 시 ~2A 버스트(배터리·태양광 사이징 주의).
- **SIM7000/7080(LTE-M/NB-IoT)**: 저전력, 소량 주기전송 최적. 커버리지는 통신사 LTE-M/NB-IoT 지원 의존. **GNSS는 모뎀 통합형**.
- ★**판단**: 결정적 이유 = **L76K 별도 GNSS**(모뎀 꺼도 GPS 독립 동작 → store-and-forward 절전의 전제). SIM7000G류는 GNSS가 모뎀 통합형이라 이 독립성이 오히려 약해진다. + Cat-1의 커버리지 단순성·처리량 여유·이미 구축된 전 스택(펌웨어·저전압 가드·BOM·CAD). 2A 버스트는 30분 주기+딥슬립+저전압 가드+배터리 장착 시험으로 관리한다. **저전력이 최우선이 되면 SIM7080G는 문서화된 껍질 대안(TinyGSM 추상화라 매크로 교체)** — 단 이는 벤치 전력예산 실측 후 결정할 일이지 이론으로 지금 바꾸지 않는다.
