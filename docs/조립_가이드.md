# Friendly Floaty 조립 가이드 (Tier 1 · 셀룰러 보틀) — 완전판

> **한 줄 요약**: 부품 도착 → 프린트(인필 ≤60%) → 펌웨어 플래시 → 벤치 테스트(배터리 장착!) → 조립(폼 160㎤·밸러스트 저중앙·안테나 상단) → 밀봉 → F1/F2·T1~T6 자가검증 → 방류 전 체크리스트. **테스트를 통과하지 못한 유닛은 방류하지 않는다.**

영문 개요는 [BUILD.md](BUILD.md). 근거 문서: [RELIABILITY](../hardware/RELIABILITY.md) · [FLOAT_STANDARD](../hardware/FLOAT_STANDARD.md) · [VERIFIED_BUILD_TEST](../hardware/VERIFIED_BUILD_TEST.md) · [PRELAUNCH_RECOVERY_CHECKLIST](../hardware/PRELAUNCH_RECOVERY_CHECKLIST.md) · [ELECTRONICS_POWER](../hardware/ELECTRONICS_POWER.md) · [firmware](../firmware/).

> ★**솔라 변형(Tier 1.5)**: 하구에 주+ 머무는 장기 잔류 임무면 배터리 단독으론 회수 전에 죽는다. 원통 감싸는 박막 셀로 보고 수명을 늘리는 변형은 [TIER1_5_SOLAR_DRIFTER](../hardware/TIER1_5_SOLAR_DRIFTER.md)를 따른다(신규 프린트 0·랩+COTS). 아래 조립에 4가지만 더한다: ①**저온 충전 컷오프(10k NTC 셀 캔 본딩, 전 모드 하드웨어 필수)** ②**비금속 RF투과 셀만**(a-Si/폴리이미드; 금속기재 금지) ③배선은 **절단-재밀봉 이음부 그로밋 재사용 → T1 재통과** ④폼 **밀도인지 +6.7 cm³/10g**로 재단. 검증은 **G5(솔라 수확)·T7(셀 장착 자기복원)·T1·F2·T6(셀 장착 RF)** 전부 통과 후 방류. 솔라는 전력이지 커버리지가 아니다(외해는 여전히 침묵). 단기 회수형이면 이 절 건너뛰고 그대로 Tier 1.

## 1. 프린트 & 폼 재단

1. 병 내경을 실측해 [`params.scad`](../hardware/cad/params.scad)의 `bottle_id`를 고치고 3부품 STL 재생성(기본값 = 1L 원형 근사).
2. **인필 ≤60%로** 브래킷·킬·회수고리 출력. (실측: 3부품 solid 부피 56.9㎤ → 60%면 ~42g)
3. 폼 재단: 병에 들어갈 **160㎤**. ★**1:1 재단 템플릿**(A4 100% 인쇄·캘리브레이션 자·두께별 적층표·레이저용 DXF 포함): [uploads.caresea.kr/seacut/hardware/foam_template.html](https://uploads.caresea.kr/seacut/hardware/foam_template.html) — 10mm EVA면 **칼라당 3장×2**. 원본은 `hardware/cad/foam_collar_2d.scad`(병 규격 바꾸면 재출력). 블록형 대체는 80×40×50mm.
4. **폼 F1 소크 테스트**(소재 자격): 폼 무게 재고 → 24h 물에 눌러 담금 → 재측정. **질량 증가 <5% 합격.** 재사용/폐부표 폼은 반드시 이걸 통과해야 함.

### 1-B. ★무프린트 경로 — 3D 프린터 없이 (칼·접착만)

프린터는 전 세계 어디에나 있는 도구가 아니다. **부품의 모양이 아니라 기능을 만들면 된다**(DIY 다양성 원칙 — 커널만 지키면 껍질은 자유):

| 프린트 부품 | 기능 | 무프린트 대안 |
|---|---|---|
| 전자부 브래킷 | 보드·배터리 고정 | 폼 칼라 채널에 보드를 끼우고 **케이블타이**. 보강은 **플라스틱 골판지(단프라)/PP 판재** ㄷ자 재단 |
| 자기복원 킬 | 무게추를 배 바닥 중앙에 | 봉돌 ~40g을 지퍼백/수축튜브에 싸 **병 바닥 안쪽 중앙에 글루건+케이블타이** |
| 회수 고리 | 건질 손잡이 | 병목에 **파라코드 짧은 매듭 고리**(느슨한 올가미 금지 — 야생동물 얽힘) |

- 부력: 프린트 ~35g이 빠져 오히려 여유 증가. 단 **T2(침수부양)·T4(자기복원)는 동일하게 통과**해야 Verified Build.
- ★금지 재료: 하드보드지·골판지·MDF(물 흡수=F1 탈락), 금속(RF 차단), 개방셀 스펀지. 허용: 단프라·PP/PET 판재·EVA·케이블타이·파라코드·글루건(전 세계 문구점·철물점 조달 가능).

## 2. 펌웨어 플래시 & 벤치 (조립 전!)

> ★**조립·밀봉 전에** 전자부가 완동하는지 확인한다. 밀봉 후 발견하면 다 뜯는다.

1. Arduino IDE + ESP32 core 3.0.x, 보드 "ESP32 Dev Module". 라이브러리: **lewisxhe/TinyGSM-fork**(스톡 TinyGSM 제거!), TinyGPSPlus, ArduinoHttpClient.
2. [`drifter_a7670_cat1.ino`](../firmware/drifter_a7670_cat1/drifter_a7670_cat1.ino)에서 `DEVICE_ID`(유닛마다 고유!)·`SITE_ID`·`APN`·`SERVER_HOST` 설정. 처음엔 `BENCH_HTTP 1` + `SLEEP_MINUTES 5`.
3. **18650 장착 후** USB 연결(★USB 단독 급전은 2A 피크에서 브라운아웃 → 모뎀 고장처럼 보임).
4. 시리얼 모니터(115200)로 확인: 모뎀 응답 → 망 등록(SIM APN) → **창가/야외에서** GPS fix → POST 200.
5. **v1.1 확인 포인트**: 로그에 `seq`가 웨이크마다 1씩 증가하는지, 서버가 `ts_fix`·`sample_interval_s`를 받는지, **안테나 분리로 망을 끊고 2~3주기 돌린 뒤 복구했을 때 버퍼가 몰아서 전송되는지**(store-and-forward).
6. **★GPS 전원 관문(1차)**: fix 실패 로그에 `NMEA chars=0`이 찍히면 L76K가 모뎀 전원에 의존하는 것 — [firmware/README](../firmware/README.md)의 벤치 재검증 체크리스트 1번대로 조치. `chars>0`이면 단순 미픽스(야외로).

## 3. 조립

### 3-0. ★병 준비 — 절단·재밀봉 (QA에서 잡힌 필수 공정)

표준 페트병 입구는 **Ø28~30mm**인데 폼 칼라(Ø86)·브래킷(폭 86)은 그보다 크다 — **입구로는 물리적으로 안 들어간다.** 두 가지 길:

1. **절단-재밀봉(기본, Merlino 아르노강 방식)**: 병 어깨 아래를 커터로 깨끗이 한 바퀴 절단 → 부품 삽입·고정 → 절단부를 5~10mm 겹쳐 끼우고 **실리콘 도포 + 방수 테이프 2겹(또는 열수축 밴드)** 으로 재밀봉. ★절단부가 새 누수 지점이 되므로 **T1(밀봉 침지 30분, Δm≤0.5g)을 반드시 이 상태로** 통과시킨다.
2. **광구 용기(대안)**: 입구가 넓은 HDPE/PP 밀폐 용기(FLOAT_STANDARD 소재표)를 쓰면 절단 불필요 — 재사용·견고 캠페인에 유리.

순서(아래→위): **킬(밸러스트) → 배터리 → 폼 → 보드 → 안테나**. 원칙: 무거운 건 아래·가운데, 폼은 상부 2/3, 안테나는 최상단.

1. 킬 포켓에 밸러스트 ~40g 고정(움직이면 복원성 죽음 — 실리콘/글루로 고정). ★밸러스트를 늘리면 폼도 늘어야 함(+10g → +11.7㎤).
2. 브래킷에 보드+18650 장착, 배선 정리. 건조제 1포 동봉.
3. 폼 칼라/블록 삽입(보드 채널 통과).
4. **안테나 2개(LTE·GNSS)는 병 최상단**, 금속·물과 이격. 하우징은 비금속(RF 투과) 유지.
5. QR/연락처 라벨(방수)·고시인성 테이프(수선 위치)·회수 고리(파라코드, **저프로파일** — 야생동물 얽힘 저감) 부착.

## 3-S. 솔라 랩 시공 (Tier 1.5 · 솔라 변형만)

박막 랩은 이 변형의 **유일한 핵심 공정**이자 최다 실패모드(물속 박리·엣지 침투)다. 순서:
1. **표면 전처리** — 몸통 랩 구간 탈지(IPA) → 실란 PET 프라이머 도포 → 지정 경화시간 대기.
2. **필름 부착** — 밴드 폭 44mm, 중앙 채널 34mm(`solar_band_gap`, 회수고리·QR 노출) 유지. VHB 마린으로 리딩 엣지부터 밀착, 곡면 기포 제거.
3. **★+Z 120° 필름-프리 창** — 캡 기준 각도를 맞춰 안테나·회수고리 위 120° 섹터를 필름으로 **덮지 말 것**([assembly_solar.png](../hardware/cad/assembly_solar.png) 대조). 마스킹 테이프로 창 경계 표시 후 재단.
4. **엣지 밀봉** — 리드 관통·필름 가장자리를 부틸/에폭시로 밀봉(박리·침투 차단). 배선은 **절단-재밀봉 이음부 그로밋 재사용**(§3-0) — 새 구멍 금지.
5. **부착물** — 충전 컨트롤러·NTC(셀 캔 본딩)·격리 다이오드는 기존 브래킷 리브에 지퍼타이(★무프린터 빌드는 폼 칼라 상부 채널 벽 또는 셀홀더 옆에 케이블타이+에폭시, CG 낮게), 벤트 패치는 캡에.

전개(flat) 절단 치수: 원주 ≈ π×`solar_wrap_od`(~289mm) · 축폭 44mm · 채널 34mm · 120° 필름-프리 호. ★정밀 SVG/DXF 전개 템플릿은 폼(`foam_collar_2d.scad`)처럼 후속 산출 예정.

## 4. 밀봉

계층 선택(RELIABILITY §7): **파일럿·회수형 = 캡+O링/실리콘**(재사용) 권장. 장기·거친 캠페인만 포팅(수리 불가).

1. 캡 나사산에 실리콘 얇게 → 체결. 관통부(있다면) 그로밋+실리콘.
2. ★**압력 균등**: 밀폐 병 + 직사광 + 리튬 = 내부 가압 위험(F2 셀벤팅). 통기벤트(Gore-Tex 패치)를 달거나, 회수형 단기 운용 + 차광(불투명 테이프 상부)으로 관리.
3. 밀봉 후 무게 실측 → 기록(질량 예산 대조: 총 ~205g 목표, RELIABILITY §1).

## 5. 자가검증 (통과 못 하면 방류 금지)

[VERIFIED_BUILD_TEST](../hardware/VERIFIED_BUILD_TEST.md)의 6종. 핵심 둘:

| 테스트 | 방법 | 합격 |
|---|---|---|
| **T1 밀봉 침지** | 30분 침지(20cm 수몰) → 개봉 없이 무게 | **Δm≤0.5g**(>2g FAIL)·내부 건조 |
| **T2 침수-부양(=F2)** | 하우징에 물 채우거나 공기배수량만큼 추 → 물에 | **72h 똑바로 부유** |
| T3 낙하 | 1.5m 콘크리트 낙하 → T1 재확인 | 밀봉 유지 |
| T4 자기복원 | 뒤집어 놓기 | 안테나가 위로 스스로 복원 |
| T5 냉수 가동 | 아이스팩 수조에서 수 주기 | 핑 누락 <10% |
| T6 소크 후 RF | T1 후 야외 | GPS+LTE 재취득 |

전부 통과 → 유닛 메타디렉토리에 `verified_build: true` 기록(스키마 §3). **이게 있어야 데이터로 인정된다.**

## 6. 방류 전·후

[PRELAUNCH_RECOVERY_CHECKLIST](../hardware/PRELAUNCH_RECOVERY_CHECKLIST.md) 전 항목 ✅ 후 방류:
- 접근 클래스 A/B/C 판정 → 회수 수단 확보(**C = 방류 금지**)
- count-in = count-out · 회수 마감 T+h · 정산 로그
- 첫 방류는 **접근 쉬운 소구간 + 짧은 테더**로 시작 권장(전 파이프라인 검증), 이후 자유 방류.
