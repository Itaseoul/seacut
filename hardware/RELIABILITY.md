> **한 줄 요약**: 밀봉 1L PET 병은 배수 ~1000cm³로 총질량(추정 ~199–207g) 대비 예비부력비 약 4.8배라 정상 시 절대 안 가라앉지만, 진짜 실패모드인 밀봉 누수 침수(순부력 −97gf) 를 폐쇄셀 폼 160cm³(6.4g) 로 역전시켜 "침수돼도 뜨는" 회수-우선 설계를 성립시킨다.

## 0. 기준·규약 (정직 가드)

- **유체 기준**: 담수 하천 ρ_water = 1.0 g/cm³. 부력 = 배수부피(cm³) × 1.0 = 배수질량(g). 힘 단위는 **gf(gram-force)** 로 표기해 질량(g)과 직접 비교. (하구 기수 ρ≈1.005–1.02는 부력을 약간 **키우므로** 담수 기준이 보수적 = 안전측.)
- **지표 정의 통일(★필수)**:
  - **예비부력비(reserve buoyancy ratio) = 부력 ÷ 질량 = V_disp / M**. 확립사실의 "예비부력 4~6배"가 이 정의다.
  - **초과부력비(excess buoyancy ratio) = 순부력 ÷ 질량 = (V_disp − M) / M**. 별도 지표로만 병기하며 예비부력비와 혼용 금지.
- 모든 수치는 **추정(estimate)** 이며 계산식 동봉. 현장 실측 아님.
- 통신은 **셀룰러(LTE Cat.1 bis)** — 위성 아님. 위치·상태는 **물리 계산/정수역학** 결과이지 학습형 예측·정확도%가 아님. 운영 대원칙은 **회수-우선**(기기를 하천에 버리지 않음).

## 2. 밀봉 정상 상태 — 예비부력비 & 건현

밀봉된 1L 병 배수부피 **V_disp ≈ 1000 cm³**.

- **부력** F_b = 1000 cm³ × 1.0 = **1000 gf**
- **예비부력비 = V_disp / M = 1000 / 207 ≈ 4.83×** (M_dry 199g 기준 5.03×) — 확립사실 4~6배 범위 내.
- (참고) **초과부력비 = (1000 − 207)/207 ≈ 3.83×** — 별도 지표, 예비부력비와 구분.

**건현(freeboard) 개략** — 수직(안테나 위) 자세 가정, 내경 Ø90mm(params `bottle_id=90`) → 단면적 A = π(4.5)² = **63.6 cm²**, 유효 병 길이 L ≈ 1000/63.6 ≈ **15.7 cm**.
- 흘수 d = 207/63.6 ≈ **3.3 cm**, **건현 ≈ 15.7 − 3.3 ≈ 12.4 cm** (추정).
- ★실제 1L 병은 넥·테이퍼로 균일 원통이 아니므로 건현은 근사(추정). 높은 건현 = 큰 풍압면적 + 세장형 수직체 = 전도 경향 → 자기복원 킬 필수(§5).

## 3. ★침수(flooded) 상태 — 순부력 (폼 없음)

밀봉 누수로 병 공기가 사라지면 **1000cm³ 배수는 소멸**하고, 각 고형부품은 **자기 재질부피만큼만** 물을 밀어낸다. 순부력 = Σ(재질 배수부피) − Σ(질량).

| 부품 | 질량(g) | 밀도(g/cm³) | 배수부피(cm³) | 순부력(gf) |
|---|---|---|---|---|
| PET 병 플라스틱 | 28 | 1.38 | 20.3 | −7.7 |
| 보드(FR4+부품, 틈새 물참) | 20 | ~1.9 유효 | 10.5 | −9.5 |
| 18650(밀봉 스틸캔, 전부피 배수) | 48 | — | 18.5¹ | −29.5 |
| 밸러스트 스테인리스 | 40 | 7.9 | 5.1 | −34.9 |
| 프린트 PLA | 35 | 1.24 | 28.2 | −6.8 |
| 안테나(플라+금속) | 8 | ~2.0 | 4.0 | −4.0 |
| 실런트 실리콘 | 8 | 1.10 | 7.3 | −0.7 |
| 배선·기타 | 12 | ~1.5 | 8.0 | −4.0 |
| **합계** | **199** | | **101.9** | **−97.1 gf** |

¹ 18650 부피 = π(0.93)²×6.8 = 18.5 cm³ (Ø18.6×68mm, params `cell_d=18.6 / cell_l=68`).

**침수 순부력 ≈ −97 gf (추정, 음수) → 침몰 확정.** 이것이 진짜 실패모드다: 배터리(−29.5)와 밸러스트(−34.9)가 적자의 66%. **폐쇄셀 폼으로 이 −97gf를 상쇄+여유 확보해야 "침수돼도 뜨는" 설계 성립.**

## 4. ★폐쇄셀 폼 1차 부력 사이징 (정본 방법 = 1.25 예비부력비 기준식)

**폼 가정**: 폐쇄셀 EVA/EPE, ρ_foam ≈ **0.04 g/cm³** (범위 0.03–0.05). 폐쇄셀이라 침수돼도 흡수 없이 전부피 배수. 폼 순부력 계수 = (1.0 − 0.04) = **0.96 gf/cm³**.

**정본 사이징식** (고형부품의 자체 배수 Σ_solid를 반드시 포함 — 이를 무시하면 과대 사이징):
```
[ Σ고형배수 V_solid + V_foam ] ≥ reserve_target × [ M_dry + ρ_foam·V_foam ]
```
**+25% (reserve_target = 1.25) 대입** (M_dry=199, V_solid=101.9):
```
101.9 + V_foam ≥ 1.25 × (199 + 0.04·V_foam)
101.9 + V_foam ≥ 248.75 + 0.05·V_foam
0.95·V_foam ≥ 146.85
V_foam ≥ 154.6 cm³
```
**+20% (1.20)**: V_foam ≥ 142 cm³. **+30% (1.30)**: V_foam ≥ 167 cm³.

→ **설계 채택 V_foam = 160 cm³** (흡수수분 여유 포함, +27.5% 수준).
- 폼 질량 = 0.04×160 = **6.4 g** (총질량 199→205.4g)
- 침수 배수 = 101.9+160 = 261.9 cm³ → **침수 순부력 = 261.9 − 205.4 = +56.5 gf**, **침수 예비부력비 = 261.9/205.4 = 1.275 (+27.5% 초과부력) → PASS**.

> **각주(안전율 상향 옵션, 정본 아님)**: 고형부품의 자체 배수(82~102cm³)를 전혀 신뢰하지 않고 "고형질량 전체를 폼만으로 상쇄"하는 보수적 방법을 쓰면 ~245cm³가 나온다. 이는 방법론적으로 덜 정밀(고형 자체배수 무시)하며 **정본은 위 155cm³→채택 160cm³**다. 245cm³는 극한 안전율이 필요한 특수 캠페인에서만 옵션으로 상향.

**폼 형상 옵션** — ★폼은 **params.scad에 없는 Tier 1 BOM 비항목·신설 부품**이다(아래 §6 참조). 치수는 신설 설계값:
- **환형 칼라(collar)**: 외경 ~Ø86mm(병 내경 90 안에 여유 끼움), 내부 페이로드 채널 70×40mm 컷아웃. 단면적 = π(4.3)² − (7.0×4.0) = 58.1 − 28.0 = **30.1 cm²** → 높이 h = 160/30.1 ≈ **53 mm**(상·하 2×27mm 분할 권장).
- **단순 블록**: EVA 80×40×50mm ≈ 160 cm³, 병 곡률에 맞춰 모서리 라운드.

**재활용 옵션(정직한 트레이드오프)**:
- **폐부표/폐EVA 스크랩**: 1차 부력재 재사용 가능(Sea Cut 폐부표 재활용 서사와 시너지). ★사용 이력 폼은 셀 파손·물먹음으로 밀도 상승 위험 → **로트별 24h 침수 후 질량증가 <5% 검수 필수**.
- **재활용 HDPE(Precious Plastic)**: ρ≈0.95 → 순부력 0.05 gf/cm³뿐(거의 중립). **부력재 부적합**, 구조 브래킷/킬/회수고리 등 **구조부품**으로만 활용이 정직. 부력은 반드시 저밀도 폐쇄셀 폼이 담당.

## 5. 자기복원(self-righting) & 밸러스트-폼 트레이드오프

**원리**: 부양 시 무게중심 CG가 부력중심 CB보다 **아래**여야 복원모멘트 발생(안테나 위로 세움). 밸러스트 스테인리스 30–50g을 **병 최하부(킬 포켓, params `ballast_l=55 / ballast_w=22`)** 에 집중, 폼 칼라는 상부 2/3에 배치.

**★밸러스트-폼 트레이드오프 유도식(정정)**: 1.25 예비부력비를 유지하려면 밸러스트 Δm(g) 추가 시 필요한 추가 폼:
```
ΔV_foam = ( reserve_target·Δm − Δm/ρ_ballast ) / ( 1 − ρ_foam )
```
- **Δm = +10g, ρ_ballast=7.9, ρ_foam=0.04**: ΔV_foam = (1.25×10 − 10/7.9) / (1 − 0.04) = (12.5 − 1.27) / 0.96 ≈ **+11.7 cm³**.
- 즉 밸러스트 +10g마다 **폼 약 +11.7cm³** 필요(명시적 트레이드오프). **권장 밸러스트 = 40g**, 여유가 필요하면 35g로 낮추면 폼 여유↑.

**CG/CB 개략(추정)**: 밸러스트 40g@바닥(y≈1cm), 배터리 48g@중하(y≈5cm), 보드 20g@중(y≈8cm), 폼6.4g+안테나8g@상(y≈13cm) → 가중 CG ≈ 5.1cm. 침수(고흘수) 상태에선 CB 상승으로 수직 복원 유리. **최소 건현 권장 ≥ 30mm**(밀봉 정상 흘수 기준). ★밀봉이 온전하면 월파해도 침수 아님 — 폼은 **밀봉 실패 시 생존** 보장용(2차 안전망).

## 6. 파라메트릭 CAD 반영안 (params.scad — 실존 파일)

★`hardware/cad/params.scad` 는 **실재**하며 다음 값이 이미 정의됨(단일 진실원천):
`bottle_id=90`, `wall=2.4`, `clr=0.4`, `board_l/w/h=68/34/12`, `cell_d/l=18.6/68`, `ballast_l/w=55/22`.
이 값들을 근거로 아래 부력·폼 변수를 **추가**한다. ★폼 칼라 치수(Ø86·채널70×40)는 params에 없던 **신설 부품**이므로 새 변수로 도입한다.

```openscad
// --- flotation & hydrostatics (NEW; estimates; verify per build) ---
foam_vol             = 160;   // cm^3 closed-cell foam (flooded reserve, non-BOM new part)
foam_density         = 0.04;  // g/cm^3 EVA/EPE estimate (0.03–0.05)
reserve_ratio_target = 1.25;  // flooded excess: (disp/mass) >= this  (PASS>=1.20)
sealed_reserve_min   = 3.0;   // sealed: 1000cm3/mass >= this (design ~4.83)
freeboard_min        = 30;    // mm minimum calm-water freeboard
ballast_mass_g       = 40;    // ★tuning var; +10g needs +~11.7cm^3 foam
// --- foam collar geometry (NEW part, not derived from bottle wall) ---
foam_od     = 86;             // mm, fit inside bottle_id=90 with margin
foam_bore_l = board_l + 2;    // 70 mm payload channel (board_l=68)
foam_bore_w = board_w + 6;    // 40 mm (board_w=34)
foam_area   = PI*pow(foam_od/2,2)/100 - (foam_bore_l*foam_bore_w)/100; // ~30.1 cm^2
foam_collar_h = foam_vol / foam_area;   // ~53 mm; split 2×27
```
※ 실제 폼은 3D프린트가 아니라 **EVA 시트 절단/열성형**으로 제작 — .scad는 절단 지그·치수 산출용 마스터.

> **정직 가드 각주**: 담수(ρ=1.0) 기준·셀룰러(LTE, 위성 아님)·물리 정수역학 계산(학습형 예측·정확도% 아님)·회수-우선(기기 안 버림)·현장 실측 날조 금지. 모든 수치는 **추정(estimate)** 이며 계산식과 함께 제시. 예비부력비=부력/질량, 초과부력비=순부력/질량으로 정의를 분리 표기.
