> **한 줄 요약**: T1~T6 6종 벤치·현장 테스트를 전부 PASS해야 유닛에 `verified_build=true` 를 부여하며, 1순위는 밀봉 누수(T1)와 침수돼도 뜨는 폐쇄셀 폼 예비부력(T2)이다.

> **정직 가드 고지**: 모든 임계값은 담수(ρ≈1.0) 기준, 통신은 셀룰러(LTE Cat.1 bis, 위성 아님). 위치·상태는 물리 계산 결과이지 학습형 예측 아님. 본 문서는 절차·임계값·양식만 규정하며 **현장 실측 수치는 담지 않는다(날조 금지)**. 표의 질량·부력은 모두 **추정(estimate)**. 대원칙은 **회수-우선**.

> **연계**: 자가 공수 부력체의 성능 자격(F1 소크·F2 침수-부양)은 [FLOAT_STANDARD.md](FLOAT_STANDARD.md)에 규정되며, 여기 **T1(밀봉 누수)·T2(72h 침수-부양)** 와 연동된다. 부력체가 F1·F2를 통과해야 "Verified Float".

## 0. 왜 이 순서인가 (설계 근거)

밀봉 1L PET 병은 배수 ~1000cm³ → 부력 ~1000gf. 총질량 추정 **M_wet ≈ 207g**(밸러스트 40g 기준, RELIABILITY.md와 통일) 대비 **예비부력비 = 1000/207 ≈ 4.83× (부력/질량)** — 정상 시 안 가라앉음.
**진짜 실패모드 = 밀봉 누수 → 침수 → 공기 상실 → 고형부품(배터리·밸러스트, 물보다 무거움) 순부력 −97gf → 침몰.**
따라서 검증 1순위는 **밀봉(T1)** 과 **침수돼도 뜨는 폐쇄셀 폼 예비부력(T2)**. 폼 사이징은 정본 1.25 예비부력비식(고형 자체배수 101.9cm³ 포함) 기준 **≥155cm³ → 채택 160cm³**(RELIABILITY.md §4).

## 1. 테스트 항목·필요 장비

|---|---|---|---|
| T1 | 밀봉 침지(누수) | F1 누수 검출 | 투명 대야, 물, 주방저울(0.1g), 종이타월, 타이머 |
| T2 | 72h 부유·월파(swamp) | 침수돼도 뜨는가(폼 예비부력) | 물통/욕조, 봉수, 스포이트/주사기, 카메라 |
| T3 | 낙하·충격 | 투척·회수 낙하 시 밀봉 유지 | 줄자, 콘크리트 바닥, T1 세트 |

### T1 — 밀봉 침지(누수) 테스트 ★1순위
**절차**: ① 조립·밀봉 유닛 건조질량 m₀(0.1g) → ② 20cm 깊이 완전 수몰 30분 강제 침지 → ③ 물기 완전 제거 후 m₁ 재측정 → ④ 병 내부 응결·수적, 그로밋/실런트 이음부 육안 점검.

| 판정 | 조건 |
|---|---|
| ✅ PASS | Δm = m₁−m₀ **≤ 0.5 g** AND 내부 물방울 0 AND 내벽 응결 없음 |
| ⚠ 재작업 | 0.5g < Δm ≤ 2.0g 또는 미세 응결 → 실런트 재도포 후 재시험 |
| ❌ FAIL | Δm > 2.0g 또는 고인 물 확인 → **방류 금지** |

> 근거: 누수 2g은 이미 침수 진행, 전자부 단락 위험. 저울 0.1g 분해능으로 0.5g 문턱 판독 가능.

### T2 — 72h 부유·월파(swamp) 테스트 ★핵심
**절차**: ① 정상 밀봉으로 72h 연속 부유, 안테나부(병목) 상시 수면 위 유지·6h 간격 사진 → ② 밀봉 의도 개방, 병 내부 물 **500mL 강제 주입**(누수 최악 모사) 후 재투입 → ③ 폐쇄셀 폼 예비부력만으로 떠 있는지 관찰.

| 판정 | 조건 |
|---|---|
| ✅ PASS | (a) 72h 안테나부 상시 노출 유지 AND (b) 500mL 주입에도 **유닛 수면 잔류**(전부 수몰 안 됨) |
| ⚠ 조건부 | 500mL에서 안테나 잠기나 유닛 수면 근처 부유 → 폼 배수량 증량 후 재시험 |
| ❌ FAIL | 500mL 주입 시 **완전 침몰** → 폼 부족, 정본 사이징 재확인(≥155cm³) 후 재시험, **방류 금지** |

> 근거: 정본 사이징(폼 160cm³)은 침수 예비부력비 1.275를 보장. "침수돼도 뜬다"가 회수-우선 설계의 물리적 보증. ★고형부품 자체배수(101.9cm³)를 포함해 계산했으므로 폼 요구량은 245cm³가 아닌 160cm³다.

### T3 — 낙하·충격 테스트
**절차**: ① 밀봉 유닛 콘크리트 위 **1.5m 3회 자유낙하**(병목/밸러스트/측면 각 방향) → ② 낙하 후 즉시 **T1 침지 30분 재수행**.

| 판정 | 조건 |
|---|---|
| ✅ PASS | 병·인서트 균열 없음 AND 낙하 후 T1 Δm ≤ 0.5g |
| ❌ FAIL | 균열/밀봉 파손 또는 낙하 후 Δm > 0.5g → 방류 금지 |

### T4 — 자기복원(self-righting) 테스트
**절차**: ① 물 위에서 완전히 뒤집어(안테나부 아래로) 손 뗌 → ② 초 단위 복원시간·최종 정지 자세 사진 → ③ 5회 반복(초기 각도 다양화).

| 판정 | 조건 |
|---|---|
| ✅ PASS | 5회 모두 **≤ 5초** 자동 복원, 최종 **안테나부 수면 위** 정렬 |
| ⚠ 튜닝 | 5–10초 소요 → 밸러스트 하중/위치 튜닝(저중심 강화, +10g→폼 +11.8cm³ 트레이드오프 유의) |
| ❌ FAIL | 복원 실패 또는 안테나부 수면 아래 정착 → 밸러스트 재튜닝, 방류 금지 |

> 근거: 세장형 수직체는 복원성이 낮아(tender, 실패모드 F6) 옆으로 뜨기 쉬움. 복원팔(GM) 정량은 계산 초기값만 주고 실동작은 수조 관찰로 확정. 안테나 수면 위가 아니면 GPS·LTE 둘 다 사망.

### T5 — 냉수 가동 지속 테스트
**절차**: ① **4°C 냉수(아이스팩+물, 온도계 확인)** 부유 → ② 최소 6h(가능 시 12h) 전송 유지, 서버 수신 로그로 간격 점검 → ③ 시작·종료 배터리 전압 로그(가능 시).

| 판정 | 조건 |
|---|---|
| ✅ PASS | 4°C에서 **6h 전송 누락률 ≤ 10%** AND 전송 주기 유지 |
| ⚠ 주의 | 10–25% 누락 → Tier 2 저온 충전보호(NTC/JEITA)·벌크캡 검토 |
| ❌ FAIL | 25% 초과 누락 또는 전송 중단 → 저온 대책 없이 겨울 방류 금지 |

> 근거: 저온에서 리튬셀 용량·전류능력 저하 → 모뎀 **~2A 송신 피크**(BOM.md 명시) 브라운아웃 위험. ★반드시 배터리 장착 상태로 시험(USB 전원은 피크에서 브라운아웃, 모뎀 고장으로 오인). ※겨울 결빙 시 동결팽창(F5)은 별도 육안 검사로 병행.

### T6 — 소크 후 RF(GPS/LTE) 재취득 테스트
**절차**: ① T2 직후 또는 30분 침지 직후 실외 개활지 이동 → ② 전원 사이클 후 **콜드 스타트** TTFF·LTE attach·첫 서버 수신까지 측정 → ③ 3회 반복.

| 판정 | 조건 |
|---|---|
| ✅ PASS | 소크 후 **GPS 최초 측위 ≤ 5분** AND **LTE 첫 서버 수신 ≤ 10분**(콜드 스타트, 개활지) |
| ⚠ 주의 | GPS 5–10분 → 안테나 습기·수면 노출 재점검(F4 결로 의심) |
| ❌ FAIL | 10분 내 측위 실패 또는 서버 미수신 → 안테나 침수/실링 결함 의심, 방류 금지 |

> 근거: 온보드 L76K GNSS는 모뎀 AT GNSS가 아닌 별도 UART(NMEA/TinyGPSPlus). 셀룰러이며 위성 업링크 아님 — 음영·실내 측위 지연은 정상.

---

### T7 — 셀 장착 자기복원 (Tier 1.5 솔라 변형 전용, = T4 재시험)
**절차**: 솔라 페이로드(박막 필름 + 충전 컨트롤러 + NTC + 배선)를 전부 장착하고 폼을 `foam_vol_solar`(~186cm³)로 키운 상태에서 **T4를 그대로 재수행**. ★밸러스트는 **40g 불변**(솔라 무게 상쇄로 감량 금지 — CG 상승, 자기복원 사망).

| 판정 | 조건 |
|---|---|
| ✅ PASS | 5회 모두 ≤5초 자동 복원, 최종 안테나부 수면 위 |
| ⚠ 튜닝 | 5–10초 → 밸러스트 **위치**(질량 아님) 재튜닝 |
| ❌ FAIL | 복원 실패 또는 안테나 수면 아래 → 방류 금지 |

> 솔라 변형은 T7에 더해 **G5(솔라 수확 특성화)·T1(배선 그로밋 장착 소크)·F2(폼 `foam_vol_solar`)·T6(셀 장착 RF 재취득)** 를 추가 통과해야 한다. 상세·손익분기는 [TIER1_5_SOLAR_DRIFTER.md §4](TIER1_5_SOLAR_DRIFTER.md).

---

**Verified Build 최종 판정**: T1~T6 **전부 PASS** → `verified_build=true` 스탬프(체크리스트/스키마 연동). 하나라도 FAIL이면 방류 불가. ★**솔라 변형(Tier 1.5)은 추가로 G5+T7+T1+F2+T6(셀 장착)를 통과해야** `verified_build`.
