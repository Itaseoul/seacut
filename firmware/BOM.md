# SEA:CUT 드리프터 부품표(BOM) — ESP32 + GPS + LTE-M  〔DEPRECATED·아래 내용 신뢰 금지〕

간헐 전송("fix→POST→sleep") 드리프터 1대 기준. 가격은 국내 소매 대략치(원).

## 핵심 전자부

| 부품 | 예시 모델 | 단가(원) | 비고 |
|---|---|---|---|
| MCU | ESP32 (베어 모듈 권장, 데브보드는 시연용) | 4,000~7,000 | deep sleep 10µA대. NodeMCU 데브보드는 기생소모 큼 |
| GPS 모듈 | u-blox NEO-6M / NEO-8M | 6,000~12,000 | UART. 백업전원으로 핫스타트 절약 |
| 셀룰러 모뎀 | SIMCom SIM7080G (LTE-M/NB-IoT) | 12,000~22,000 | 게이트웨이 불필요. PSM 저전력 |
| USIM | 통신사 IoT 요금제 | 월 2,000~4,000 | SKT/KT LTE-M |
| 전원 게이팅 | P-ch MOSFET + 저항 | 1,000 | GPS·모뎀 슬립 시 완전차단 |
| 배터리 | 리튬 18650 2,000~3,000mAh | 3,000~6,000 | 방수 홀더 |
| (선택) 태양광 | 5V 1~2W 소형 패널 + TP4056 | 5,000~9,000 | 주간 충전 → 사실상 무한 |
| 안테나 | GPS 액티브 + LTE 안테나 | 3,000~6,000 | ★수면 위로 노출 필수 |

**전자부 소계: 약 3.3만~6.9만원/대** (태양광 포함 시 상단).

## 하우징·부력

| 부품 | 비고 | 단가(원) |
|---|---|---|
| 방수 케이스 | IP68, 안테나 관통부 방수 | 5,000~15,000 |
| 부력체 | 폐부표재 재활용(사업 컨셉과 정합) | 재활용 0~ |
| 무게추·날개 | 흘수 조정(수면상 최소·수면하 최대) | 2,000~ |
| (선택) 생분해 프레임 | 유실 시 환경부하 저감 | 별도 |

## 총계

- **전자부 + 하우징: 대당 약 8만~15만원** (수량·태양광·케이스 등급에 따라).
- 앞서 조사한 독일만 컴팩트 드리프터($300)에서 위성을 LTE-M으로 대체해 절반 이하.

## LoRa 대안(참고)

| 항목 | LTE-M(본 BOM) | LoRa |
|---|---|---|
| 모뎀 단가 | 1.2~2.2만 | 0.4~0.8만 (더 쌈) |
| 인프라 | 불필요(기지국) | ★게이트웨이 필요 3~15만/대 + 설치 |
| 통신비 | 월 소액 | 0(자체 게이트웨이) |
| 적합 | 소수·연안·PoC | 게이트웨이 1대 + 드리프터 대량 |

소하천 PoC는 LTE-M 권장. 붐 고정 구역에 게이트웨이 세우고 대량 운용하는 단계에서 LoRa 전환.

## 통신 요건

- 낙동강 하구·연안은 LTE 커버리지 안 → 위성 불필요.
- 안테나는 반드시 수면 위. 물에 잠기면 GPS·LTE 둘 다 죽음.
- 전송 주기 5분이면 태양광 소형 패널로 상시 운용 가능.
